# Mapeamento do Informe de Rendimentos x Eventos S-5002 e S-2501 do eSocial - GT-Piloto

## Sobre o Projeto

Este repositório contém o documento de **mapeamento** entre os eventos **S-5002 - Imposto de Renda Retido na Fonte por Trabalhador** e **S-2501 - Informações de Tributos Decorrentes de Processo Trabalhista** do e-Social (versão S-1.3) e o **Comprovante de Rendimentos Pagos e de Imposto sobre a Renda Retido na Fonte (Informe de Rendimentos)**.

## Contexto

Este documento foi construído pelas empresas de software participantes do GT-Piloto do eSocial, seguindo a Instrução Normativa RFB nº 2.060, de 13 de dezembro de 2021.

## Estrutura do Repositório

```
├── modelos/
│   └── Comprovante_Rendimentos_x_eSocial.md  # Documento principal com mapeamentos
└── README.md                                 # Este arquivo
```

## Conteúdo do Documento

O documento principal apresenta:

- **Mapeamento detalhado** dos grupos de campos do Informe de Rendimentos
- **Regras de preenchimento** conforme os eventos S-5002 e S-2501
- **Cenários específicos** para diferentes tipos de rendimentos
- **Códigos de referência** do eSocial
- **Exemplos práticos** em XML

## Grupos de Mapeamento

1. **FONTE PAGADORA PESSOA JURÍDICA OU PESSOA FÍSICA**
2. **PESSOA FÍSICA BENEFICIÁRIA DOS RENDIMENTOS**
3. **RENDIMENTOS TRIBUTÁVEIS, DEDUÇÕES E IMPOSTO SOBRE A RENDA RETIDO NA FONTE**
4. **RENDIMENTOS ISENTOS E NÃO TRIBUTÁVEIS**
5. **RENDIMENTOS SUJEITOS A TRIBUTAÇÃO EXCLUSIVA (RENDIMENTO LÍQUIDO)**
6. **RENDIMENTOS RECEBIDOS ACUMULADAMENTE**
7. **INFORMAÇÕES COMPLEMENTARES**

## Como Usar

1. Acesse o documento principal em `modelos/Comprovante_Rendimentos_x_eSocial.md`
2. Consulte os mapeamentos específicos para cada cenário
3. Utilize os códigos de referência para implementação
4. Consulte os exemplos XML no apêndice
